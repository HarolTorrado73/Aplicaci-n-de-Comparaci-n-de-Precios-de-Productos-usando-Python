# Uso Responsable

Este proyecto se creó con fines educativos para demostrar integración de microservicios.

## Principios de uso

- Úsalo de forma ética y legal.
- Respeta privacidad y protección de datos.
- Evita desplegarlo con datos personales reales.
- No utilices la app para prácticas fraudulentas o engañosas.

## Limitaciones

- No es un sistema de pricing comercial listo para producción.
- No ofrece garantías de disponibilidad, precisión o seguridad total.
- Requiere endurecimiento adicional para entorno empresarial.

## Recomendaciones de cumplimiento

- Anonimizar datos de prueba.
- Registrar consentimiento cuando se recopilen datos.
- Cumplir normativas locales aplicables (por ejemplo, protección de datos).
